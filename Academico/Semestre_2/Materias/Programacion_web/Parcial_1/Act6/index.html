Siglas

OSI (Open Systems Interconnection)

Modelo de referencia para la comunicación entre sistemas de red.

IP (Internet Protocol)

Representa la dirección lógica de un dispositivo; está formada por 4 números en un rango de 0 a 255.

TCP (Transmission Control Protocol)

Protocolo de control de transmisión orientado a la conexión.

UDP (User Datagram Protocol)

Protocolo sin conexión que envía datagramas sin verificar su entrega.

SSH (Secure Shell)

Protocolo para acceso seguro a sistemas remotos.

FTP (File Transfer Protocol)

Protocolo para la transferencia de archivos entre sistemas en una red.

SMTP (Simple Mail Transfer Protocol)

Protocolo para la transferencia de correo electrónico en formato simple.

POP (Post Office Protocol)

Servicio para acceder a la oficina postal virtual de correo electrónico.

DHCP (Dynamic Host Configuration Protocol)

Protocolo que asigna dinámicamente direcciones IP a dispositivos en una red.

DNS (Domain Name System)

Protocolo que traduce nombres de dominio en direcciones IP.

HTTP (Hyper Text Transfer Protocol)

Protocolo de transporte de archivos en formato HTML (páginas web).

ARP (Address Resolution Protocol)

Protocolo que asocia direcciones IP con direcciones físicas MAC.

MAC (Media Access Control)

Dirección física única asignada a cada dispositivo de red.

LAN (Local Area Network)

Red de área local que conecta dispositivos dentro de un espacio limitado.
